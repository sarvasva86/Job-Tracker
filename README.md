# 💼 Job Tracker

A full-stack job tracking web application built with React, Node.js, Express, and PostgreSQL.

Users can register, log in securely using JWT authentication, and manage their job applications with full CRUD functionality.

## ✨ Features

- 🔐 JWT Authentication (Register/Login)
- 🛡 Protected Routes (Frontend + Backend)
- ➕ Add job applications
- ✏ Edit job details
- ❌ Delete jobs
- 📊 Status tracking (Applied, Interview, Offer, Rejected)
- 👤 User-specific data isolation
- 🎨 Clean dashboard UI
